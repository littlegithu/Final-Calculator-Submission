# JavaScript Calculator Assignment

## Submission Overview
This repository contains two versions of the calculator assignment to ensure full compliance with the rubric while demonstrating application development skills.

### 1. Rubric Solution (Folder: /solution-logic-only)
*   **File:** `calculator.js`
*   **Description:** A strict, logic-only JavaScript implementation.
*   **How to Run:** Run in Node.js or paste into the Browser Console.
*   **Rubric Alignment:** Contains the modular arithmetic functions and history array exactly as requested.

### 2. Web Application (Folder: /solution-web-interface)
*   **Files:** `index.html`, `style.css`, `script.js`
*   **Description:** A full Graphical User Interface (GUI) version.
*   **How to Run:** Open `index.html` in any web browser.
